Transifex Appliance 1.0 (Proteus) Release

Date Fri 15 October 2010 Tags rpath

A newer version of the Transifex Appliance has been released with all the latest, juicy bits! In fact, this release is the mother of all releases with so many new features and goodies that it would take me a while to jot them down here! A few impressive nuggets are:

  • Brand New Translation Storage Engine
  • QT (TS) translation file format support
  • Source language file tracking over HTTP
  • Translation file metadata auto-updating
  • Clone language
  • Non-English Source Languages
  • Lotte: Multiple Source Languages, Suggestions, Keyboard Shortcuts, Translation Memory
  • Application Programmable Interface

Can you believe that Transifex has reached the 1.0 mark???
